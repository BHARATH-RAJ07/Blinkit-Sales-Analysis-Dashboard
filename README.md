# Blinkit Sales Analysis Dashboard

## Project Overview
The Blinkit Sales Analysis Dashboard is an interactive data visualization project built using Tableau to analyze and understand Blinkit's retail sales performance. The dashboard consolidates key business metrics and provides actionable insights into sales trends, customer preferences, and outlet performance across different dimensions such as outlet size, location type, product categories, fat content, and time.

This project is designed to support data-driven decision-making by helping stakeholders identify high-performing segments, growth opportunities, and areas requiring improvement.

---

## Objectives
- Analyze overall sales performance and customer ratings
- Understand sales contribution across different outlet sizes and locations
- Identify top-performing and underperforming item categories
- Examine customer preference based on fat content
- Track year-wise sales growth and trends

---

## Key Metrics
- Total Sales: Represents the overall revenue generated across all outlets
- Average Sales: Indicates the average sales value per item
- Item Count: Total number of items sold
- Average Rating: Reflects customer satisfaction across products

---

## Dashboard Visual Insights
- Year-wise Total Sales Trend: Displays sales growth patterns over the years and highlights peak and low-performing periods
- Sales by Item Type: Compares revenue contribution from different product categories
- Outlet Count by Size: Shows distribution of small, medium, and large outlets
- Sales by Outlet Location Type: Analyzes sales performance across Tier 1, Tier 2, and Tier 3 locations
- Fat Content Distribution: Highlights customer preference between low-fat and regular-fat products
